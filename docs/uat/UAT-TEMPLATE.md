## Post-run cleanup checklist

Complete this checklist before merging the UAT branch back to functional-test.

**Test repo (workflow) — confirm final state matches baseline:**
- [ ] `ai_provider` restored to baseline value
- [ ] `ai_api_key` references the correct secret (not an inline value)
- [ ] `ai_model` references the correct variable (not an inline value)
- [ ] `diff_lines` removed (or set to intended value, not `"10"`)
- [ ] `review_criteria` removed (or set to intended production value)
- [ ] `custom_rules` removed (or set to intended production value)
- [ ] Action ref is the intended post-UAT ref (not the UAT branch)
- [ ] Push final workflow state to test repo PR branch

**Test repo (secrets/variables) — confirm restored:**
- [ ] No invalid/placeholder API key left in secrets
- [ ] Any temporarily modified variables restored
- [ ] Note any variables changed specifically for UAT (e.g. model version) with their revert deadline

**Test PRs:**
- [ ] Any PR opened solely for a single test scenario has been closed or reviewed for intended merge
- [ ] No test PRs left open with intentionally bad code in them (sensitive data strings, invalid config)

**UAT branch:**
- [ ] All scenario results recorded (no empty Result fields)
- [ ] Result summary table updated with final counts
- [ ] Overall result set to PASS / FAIL / PARTIAL
- [ ] Sign-off section completed
- [ ] Commit and push the completed run record before merging
